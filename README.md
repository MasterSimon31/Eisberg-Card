# Eisberg – Custom Card Pack für MD Pro 3

🇩🇪 Ein selbstgebauter WASSER-Archetyp (8 Karten) mit eigenem Beschwörungs-Konzept:
**Tiefenbeschwörung**. Die Eisberg-Monster beschwören sich **direkt aus dem Deck**, indem
sie die obersten Karten offen verbannen, und **frieren** das Gegnerfeld mit **Eis-Zählmarken**
ein (Position sperren + Effekte annullieren).

---

## Installation (30 Sekunden)

1. In MD Pro 3: **Settings → Expansions → "Download Custom Card Pack"**
2. Diesen Link einfügen:
   ```
   https://raw.githubusercontent.com/MasterSimon31/Eisberg-Card/main/eisberg.ypk
   ```
3. Spiel **komplett neu starten**.
4. Im Deck-Editor nach **"Eisberg"** suchen – die 8 Karten sind da.

> Zum Online-Spielen müssen **beide** Spieler das Pack installiert haben.

---

## Die 8 Karten

| Karte | Typ | 🧊 | Kurz |
|---|---|---|---|
| Frostauge-Schneeeule | Effekt-Monster · WASSER | 2 | Deck-Beschwörung, legt Eis-Zählmarken, Quick-Negate |
| Gletscher-Glyphodrachen | Effekt-Monster · WASSER | 4 | Setzt Gegner-ATK auf 0, Piercing gegen eingefrorene Monster |
| Frostherz-Leviathan | Effekt-Monster · WASSER (Boss) | 6 | Schickt eingefrorene Monster in den Friedhof, End-Phase-Revive |
| Das Eismeer | Feldzauber | – | Sucht 1 Eisberg-Monster, legt Zählmarken beim Verbannen |
| Tiefenströmung | Schnellzauber | – | Millt 3, aus dem Friedhof: legt Eisberg-Zauber/Falle |
| Auftauen | Zauber | – | Mischt Verbannte zurück + zieht |
| Eisiger Hinterhalt | Falle | – | Legt Zählmarke + verhindert Angriff |
| Ewiger Frost | Konterfalle | – | Verbannt spezialbeschworene Gegner-Monster |

**Kernidee:** Beschwöre deine Monster aus dem Deck, indem du tief „tauchst" (Karten verbannst),
und friere das Gegnerfeld Stück für Stück mit Eis-Zählmarken ein.

## Eigener Eisberg-Look (optional, empfohlen)

MD Pro 3 baut Karten immer aus seinem Standard-Rahmen + Artwork zusammen. Für den vollen
dunklen **Tiefenbeschwörungs-Look** der 3 Monster gibt es deshalb ein kleines Extra:

1. **[`eisberg-overframe.zip`](eisberg-overframe.zip)** herunterladen und entpacken (3 PNGs).
2. Die 3 Dateien nach `<MD-Pro-3-Ordner>\Picture\OverFrame\` kopieren.
3. Spiel neu starten – die Monster erscheinen im eigenen dunklen Eisberg-Rahmen.

*(Rein optisch, ändert nichts am Spiel. Ohne das Extra zeigen die Monster ihr Artwork im
normalen Rahmen.)*
